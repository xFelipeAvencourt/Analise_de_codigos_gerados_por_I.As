Análise de Código Gerado por IA para Redes Programáveis

Este repositório contém os dados e resultados de um trabalho focado na análise da eficiência de códigos gerados por Inteligência Artificial (IA) aplicados a problemas de Redes Programáveis (RP).

Contexto:

A Inteligência Artificial generativa tem sido amplamente utilizada para a geração automática de códigos em linguagens como C, Python, C++ e Java, baseando-se em descrições de objetivos fornecidas pelo usuário. Embora essa tecnologia possa gerar resultados que atendem aos requisitos especificados , os programas resultantes podem apresentar erros lógicos ou problemas de desempenho.

Este problema torna-se crítico quando aplicado a dispositivos de rede que exigem segurança robusta, como firewalls. Portanto, este projeto visa investigar a viabilidade e a qualidade dessas soluções no contexto de redes.

Objetivo:

O objetivo principal é analisar as eficiências de códigos gerados por ferramentas de IA para resolver problemas específicos relacionados a Redes Programáveis (RP).

Ferramentas de IA Avaliadas:

Foi realizada uma revisão teórica para selecionar as ferramentas submetidas aos testes. A seleção baseou-se em critérios de popularidade e abordagem técnica, resultando nas seguintes IAs:

- ChatGPT

- Copilot

- DeepSeek

- Blackbox.ai

Metodologia:

A metodologia de avaliação consistiu nas seguintes etapas:

Definição de Prompts: Foram definidos três prompts com tarefas de diferentes níveis de complexidade: simples, média e complexa.

Técnicas de Abordagem: Foram utilizadas duas técnicas para informar os prompts e verificar a eficácia de resolução:

(i) Zero-shot 

(ii) Few-shot

Execução: Para cada tarefa, iniciou-se um novo chat para definição do problema conforme a abordagem escolhida.

Testes e Simulação: O código gerado foi testado em diferentes topologias de rede, simulando diversos tipos de redes.

Resultados:

Para a maioria das tarefas propostas, as IAs implementaram soluções satisfatórias logo na primeira interação.

Em alguns casos, houve a necessidade de mais interações para realizar as correções necessárias.
